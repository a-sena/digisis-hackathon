# Hjelp — Oppgave 9: Topp-3 per postnummer

To strategier — velg den du synes er mest lesbar:

**A)** Sorter listen synkende på inntekt, grupper på postnummer, og
kutt hver gruppe til 3 elementer etterpå.

**B)** Grupper først på postnummer, deretter sorter og kutt hver
gruppes liste hver for seg.

Husk at å kutte til "maks 3" må fungere selv når gruppen har færre
enn 3 elementer (uten å krasje).
